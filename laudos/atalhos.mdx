# Atalhos de Teclado

Utilize atalhos para agilizar seu fluxo de trabalho no Editor.

## Atalhos Globais

| Atalho | Funcao |
|--------|--------|
| `Cmd + K` | Buscar laudos |
| `Cmd + S` | Salvar laudo |
| `Alt` | Ativar/desativar ditado por voz |
| `Espaco` | Iniciar ditado (quando no modo de espera) |

## Formatacao de Texto

| Atalho | Funcao |
|--------|--------|
| `Cmd + B` | Negrito |
| `Cmd + I` | Italico |
| `Cmd + U` | Sublinhado |

## Navegacao

| Atalho | Funcao |
|--------|--------|
| `Alt + T` | Abrir painel de notificacoes |

<Tip>
Clique no botao **Atalhos** na barra de ferramentas do editor para ver a lista completa de atalhos disponiveis.
</Tip>

## Atalhos do Editor

| Atalho | Funcao |
|--------|--------|
| `Cmd + Z` | Desfazer |
| `Cmd + Shift + Z` | Refazer |
| `Cmd + A` | Selecionar tudo |
| `Cmd + C` | Copiar |
| `Cmd + V` | Colar |
