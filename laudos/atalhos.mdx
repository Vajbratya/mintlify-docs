# Atalhos de Teclado

Domine os atalhos do Laudos.AI para criar laudos com máxima eficiência. O sistema foi projetado para minimizar o uso do mouse e maximizar produtividade.

<Info>
**Dica Principal**: Pressione `Cmd+K` a qualquer momento para abrir a **Busca Global** - o comando mais poderoso da plataforma.
</Info>

---

## Cmd+K - Busca Global (Command Palette)

O atalho `Cmd+K` (ou `⌘K`) abre a **Busca Global**, uma ferramenta poderosa que permite:

### O que você pode fazer

<CardGroup cols={2}>
  <Card title="Buscar Laudos" icon="file-medical">
    Digite parte do título, tipo de exame ou data para encontrar laudos rapidamente
  </Card>
  <Card title="Navegar Rápido" icon="compass">
    Acesse qualquer página da plataforma sem usar o mouse
  </Card>
  <Card title="Buscar Modelos" icon="folder-open">
    Encontre templates por nome ou categoria
  </Card>
  <Card title="Ações Rápidas" icon="bolt">
    Execute comandos como "Novo Laudo", "Dashboard", etc.
  </Card>
</CardGroup>

### Como usar

<Steps>
  <Step title="Abra a Busca Global">
    Pressione `Cmd+K` (Mac) ou `Ctrl+K` (Windows/Linux) em qualquer tela.
  </Step>

  <Step title="Digite sua busca">
    Comece a digitar o que procura:
    - Nome do laudo: "TC Abdome"
    - Tipo de exame: "Tomografia"
    - Data: "05/01/2026"
    - Ação: "Novo laudo"
  </Step>

  <Step title="Navegue nos resultados">
    Use as setas ↑↓ para navegar e `Enter` para selecionar.
  </Step>

  <Step title="Feche com Escape">
    Pressione `Esc` para fechar sem selecionar.
  </Step>
</Steps>

### Exemplos de Busca

| Você digita | Resultado |
|-------------|-----------|
| `tc abd` | Laudos de TC de Abdome |
| `torax` | Laudos e modelos de tórax |
| `novo` | Ação "Criar novo laudo" |
| `dashboard` | Navega para o painel |
| `config` | Abre configurações |
| `modelo` | Lista de templates |

<Tip>
A busca é **fuzzy** - não precisa digitar exatamente. "tcabd" encontra "TC Abdome Total".
</Tip>

## Atalhos do Editor

### Criação e Salvamento

| Atalho | Função | Descrição |
|--------|--------|-----------|
| `Cmd+N` | Novo laudo | Cria um laudo em branco |
| `Cmd+S` | Salvar | Salva o laudo atual |
| `Cmd+Shift+S` | Salvar como template | Transforma o laudo em modelo reutilizável |

### Ditado por Voz

| Atalho | Função | Descrição |
|--------|--------|-----------|
| `Espaço` | Iniciar ditado | Funciona quando o editor está vazio |
| `Alt+M` | Toggle microfone | Ativa/desativa em qualquer momento |
| `Espaço` | Pausar/retomar | Durante a gravação |

### Formatação de Texto

| Atalho | Função | Toggle |
|--------|--------|--------|
| `Cmd+B` | **Negrito** | ✅ |
| `Cmd+I` | *Itálico* | ✅ |
| `Cmd+U` | <u>Sublinhado</u> | ✅ |
| `Cmd+Z` | Desfazer | - |
| `Cmd+Shift+Z` | Refazer | - |

### Assistência de IA

| Atalho | Função | Descrição |
|--------|--------|-----------|
| `Cmd+/` | Gerar sugestão | Pede à IA para completar ou sugerir texto |
| `Cmd+L` | Abrir LaudAI | Abre o assistente de IA |

## Atalhos de Navegação

### Navegação Global

| Atalho | Destino |
|--------|---------|
| `Cmd+K` | Busca Global |
| `Cmd+Shift+H` | Dashboard (Home) |
| `Cmd+Shift+E` | Editor |
| `Cmd+Shift+R` | Biblioteca de Laudos (Reports) |
| `Cmd+Shift+T` | Biblioteca de Modelos (Templates) |

### Dentro do Editor

| Atalho | Função |
|--------|--------|
| `Tab` | Próximo campo |
| `Shift+Tab` | Campo anterior |
| `Esc` | Fechar modal/diálogo |
| `Enter` | Confirmar ação |

---

## Atalhos de Ação

### Operações com Laudos

| Atalho | Função |
|--------|--------|
| `Cmd+P` | Imprimir laudo |
| `Cmd+Shift+P` | Pré-visualizar impressão |
| `Cmd+C` | Copiar texto selecionado |
| `Cmd+V` | Colar texto |

### Interface

| Atalho | Função |
|--------|--------|
| `Cmd+Shift+/` | Mostrar painel de atalhos |
| `Esc` | Fechar diálogos e modais |

## Dicas de Produtividade

<AccordionGroup>
  <Accordion title="Workflow Rápido: Novo Laudo por Voz">
    ```
    Cmd+N → Espaço → [Ditar] → Espaço → Cmd+S
    ```
    1. `Cmd+N` - Cria novo laudo
    2. `Espaço` - Inicia ditado
    3. Fale seu laudo
    4. `Espaço` - Para ditado
    5. `Cmd+S` - Salva
  </Accordion>

  <Accordion title="Workflow Rápido: Laudo com Template">
    ```
    Cmd+K → [buscar modelo] → Enter → [preencher] → Cmd+S
    ```
    1. `Cmd+K` - Abre busca
    2. Digite o nome do modelo
    3. `Enter` - Seleciona
    4. Preencha os campos
    5. `Cmd+S` - Salva
  </Accordion>

  <Accordion title="Navegação sem Mouse">
    ```
    Cmd+K → "dashboard" → Enter
    Cmd+K → "templates" → Enter
    Cmd+K → "config" → Enter
    ```
    Use `Cmd+K` para ir a qualquer lugar sem tocar no mouse!
  </Accordion>

  <Accordion title="Formatação Rápida">
    Selecione o texto e use:
    - `Cmd+B` para **negrito**
    - `Cmd+I` para *itálico*
    - `Cmd+U` para sublinhado

    Os atalhos funcionam como toggle - pressione novamente para remover.
  </Accordion>
</AccordionGroup>

## Acessando o Painel de Atalhos

Na plataforma, você pode ver todos os atalhos de duas formas:

1. **Botão "Atalhos"** na barra de ferramentas do editor
2. **Atalho `Cmd+Shift+/`** em qualquer tela

<Tip>
O painel de atalhos na plataforma sempre mostra os atalhos mais atualizados. Esta documentação serve como referência, mas a plataforma é a fonte definitiva.
</Tip>
