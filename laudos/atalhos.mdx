# Atalhos de Teclado

Comandos rapidos para aumentar sua produtividade.

<Tip>
Pressione `Cmd + Shift + /` para abrir o painel de atalhos a qualquer momento.
</Tip>

## Navegacao

| Atalho | Funcao |
|--------|--------|
| `Cmd + L` | Abrir assistente LaudAI |
| `Cmd + Shift + H` | Ir para dashboard |
| `Cmd + Shift + E` | Ir para editor |
| `Cmd + Shift + R` | Ver relatorios |
| `Cmd + Shift + T` | Ver templates |

## Editor

| Atalho | Funcao |
|--------|--------|
| `Cmd + N` | Novo laudo |
| `Cmd + S` | Salvar laudo |
| `Cmd + Shift + S` | Salvar como template |
| `Alt + M` | Ativar/desativar ditado |
| `Cmd + /` | Gerar sugestao LaudAI |

## Acoes

| Atalho | Funcao |
|--------|--------|
| `Cmd + P` | Imprimir laudo |
| `Cmd + Shift + P` | Pre-visualizar laudo |

## Visualizacao

| Atalho | Funcao |
|--------|--------|
| `Cmd + Shift + /` | Mostrar atalhos |
| `Escape` | Fechar dialogos |

## Formatacao de Texto

| Atalho | Funcao |
|--------|--------|
| `Cmd + B` | Negrito |
| `Cmd + I` | Italico |
| `Cmd + U` | Sublinhado |
