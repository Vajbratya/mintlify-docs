# Atalhos de Teclado

Comandos rápidos para aumentar sua produtividade.

<Tip>
Pressione `Cmd + Shift + /` para abrir o painel de atalhos a qualquer momento.
</Tip>

## Navegação

| Atalho | Função |
|--------|--------|
| `Cmd + L` | Abrir assistente LaudAI |
| `Cmd + Shift + H` | Ir para dashboard |
| `Cmd + Shift + E` | Ir para editor |
| `Cmd + Shift + R` | Ver relatórios |
| `Cmd + Shift + T` | Ver templates |

## Editor

| Atalho | Função |
|--------|--------|
| `Cmd + N` | Novo laudo |
| `Cmd + S` | Salvar laudo |
| `Cmd + Shift + S` | Salvar como template |
| `Alt + M` | Ativar/desativar ditado |
| `Cmd + /` | Gerar sugestão LaudAI |

## Ações

| Atalho | Função |
|--------|--------|
| `Cmd + P` | Imprimir laudo |
| `Cmd + Shift + P` | Pré-visualizar laudo |

## Visualização

| Atalho | Função |
|--------|--------|
| `Cmd + Shift + /` | Mostrar atalhos |
| `Escape` | Fechar diálogos |

## Formatação de Texto

| Atalho | Função |
|--------|--------|
| `Cmd + B` | Negrito |
| `Cmd + I` | Itálico |
| `Cmd + U` | Sublinhado |
