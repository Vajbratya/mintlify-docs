# Editor de Laudos

O Editor é a interface principal para criar, editar e gerenciar laudos médicos.

## Layout da Interface

### Barra Superior

- **Menu hambúrguer**: Acesso às configurações gerais
- **Sugerir melhoria**: Enviar feedback para a equipe
- **Buscar laudos**: Pesquisa rápida (atalho: Cmd+K)
- **Dashboard**: Retornar ao painel de controle
- **Salvar**: Salvar laudo atual
- **Tutorial**: Assistir tutorial em vídeo

### Barra Lateral Esquerda (Laudos Recentes)

Lista os últimos 15 laudos com:
- Título e data
- Tags: Voz, Auto-gerado, Modelo utilizado
- Categoria (Radiologia, etc.)

### Área Principal do Editor

#### Ferramentas de Formatação

| Botão | Função | Atalho |
|-------|--------|--------|
| **B** | Negrito | Cmd+B |
| *I* | Itálico | Cmd+I |
| U | Sublinhado | Cmd+U |
| **H1** | Título 1 | - |
| **H2** | Título 2 | - |
| Lista | Lista não numerada | - |
| Lista Numerada | Lista ordenada | - |

#### Ferramentas Especiais

- **Buscar Artigos**: Pesquisa referências científicas
- **Extrair Frases**: Extrai frases do texto para salvar
- **Crit**: Análise de achados críticos
- **Copiar**: Copia o laudo para área de transferência
- **Atalhos**: Lista de atalhos de teclado
- **Publicar**: Finaliza e publica o laudo

## Criando um Novo Laudo

<Steps>
  <Step title="Escolha o Método">
    - **Ditar laudo**: Pressione **Espaço** para iniciar ditado por voz
    - **Selecionar modelo**: Escolha um template existente
    - **Em branco**: Inicie do zero
    - **Digitar**: Comece digitando diretamente
  </Step>

  <Step title="Modelos Frequentes">
    Acesso rápido aos templates mais usados:
    - rededorabdcom
    - TORAXrede
    - abdhaoc
    - ABDTOTALHSC
    - toraxhaoc
  </Step>

  <Step title="Edite o Conteúdo">
    Use as ferramentas de formatação e a assistência de IA para completar o laudo.
  </Step>

  <Step title="Salve e Publique">
    Clique em **Salvar** ou use o salvamento automático.
  </Step>
</Steps>

## Barra de Status Inferior

- **Caracteres**: Contagem de caracteres (ex: 1.904)
- **Palavras**: Contagem de palavras (ex: 381)
- **Ditado por voz**: Atalho Alt+M
- **Salvar**: Atalho Cmd+S
- **Buscar**: Atalho Cmd+K
- **Formato**: Seletor de formato (Moderno)
- **Limpar**: Limpa o editor
- **Novo**: Cria novo laudo

## Painel de Achados Críticos

À direita do editor, o painel **Achados Relevantes** exibe:
- Monitoramento automático de achados
- Contagem de achados críticos detectados
- Botão para reanalisar achados

<Warning>
Conforme orientação do CBR, achados **críticos** exigem comunicação em até 1 hora. Achados **urgentes não críticos** devem ser comunicados em até 3 horas.
</Warning>
