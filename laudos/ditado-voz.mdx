# Ditado por Voz

O Laudos.AI possui reconhecimento de voz avancado em portugues brasileiro, com vocabulario medico especializado para radiologia.

## Como Usar

### Passo 1: Selecione um Modelo (Opcional mas Recomendado)

<Warning>
**Importante**: Para que a IA siga a estrutura e formatacao do seu modelo, selecione o template **ANTES** de iniciar o ditado.
</Warning>

Na barra inferior do editor, use a busca para encontrar seu modelo:

| Categoria | Tipo de Exame |
|-----------|---------------|
| **Todos** | Todos os modelos |
| **TC** | Tomografia Computadorizada |
| **RM** | Ressonancia Magnetica |
| **RX** | Raio-X |
| **US** | Ultrassonografia |

Clique no modelo desejado para seleciona-lo.

### Passo 2: Iniciar Ditado

1. Pressione a tecla **Espaco** quando o editor estiver vazio
2. Ou pressione **Alt + M** a qualquer momento
3. O indicador de gravacao aparecera na barra inferior

### Ditar seu Laudo

Fale naturalmente descrevendo os achados do exame:

<Note>
Exemplo: "Figado de dimensoes normais, contornos regulares, ecotextura homogenea. Vesicula biliar normodistendida, sem calculos. Vias biliares nao dilatadas..."
</Note>

### Parar Ditado

- Pressione **Espaco** ou **Alt** novamente
- O texto sera transcrito e processado pela IA
- A IA estrutura automaticamente o conteudo em formato de laudo

## Recursos de Voz

### Vocabulario Medico

O sistema reconhece termos tecnicos de radiologia:
- Anatomia e estruturas
- Patologias e achados
- Medidas e dimensoes
- Terminologia padrao

### Processamento Inteligente

Apos o ditado, a IA:
- Estrutura o texto em secoes apropriadas
- Corrige erros de transcricao
- Aplica formatacao padrao
- Sugere conclusoes baseadas nos achados

## Atalhos de Voz

| Atalho | Funcao |
|--------|--------|
| `Espaco` | Iniciar/parar ditado (editor vazio) |
| `Alt + M` | Iniciar/parar ditado (qualquer momento) |
| `Cmd + S` | Salvar laudo |
| `Cmd + K` | Buscar laudos |
| `Cmd + /` | Ver todos os atalhos |
