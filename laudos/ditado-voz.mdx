# Ditado por Voz

O Laudos.AI possui reconhecimento de voz avançado em português brasileiro, com vocabulário médico especializado para radiologia.

## Como Usar

### Passo 1: Selecione um Modelo (Opcional mas Recomendado)

<Warning>
**Importante**: Para que a IA siga a estrutura e formatação do seu modelo, selecione o template **ANTES** de iniciar o ditado.
</Warning>

Na barra inferior do editor, use a busca para encontrar seu modelo:

| Categoria | Tipo de Exame |
|-----------|---------------|
| **Todos** | Todos os modelos |
| **TC** | Tomografia Computadorizada |
| **RM** | Ressonância Magnética |
| **RX** | Raio-X |
| **US** | Ultrassonografia |

Clique no modelo desejado para selecioná-lo.

### Passo 2: Iniciar Ditado

1. Pressione a tecla **Espaço** quando o editor estiver vazio
2. Ou pressione **Alt + M** a qualquer momento
3. O indicador de gravação aparecerá na barra inferior

### Ditar seu Laudo

Fale naturalmente descrevendo os achados do exame:

<Note>
Exemplo: "Fígado de dimensões normais, contornos regulares, ecotextura homogênea. Vesícula biliar normodistendida, sem cálculos. Vias biliares não dilatadas..."
</Note>

### Parar Ditado

- Pressione **Espaço** ou **Alt + M** novamente
- O texto será transcrito e processado pela IA
- A IA estrutura automaticamente o conteúdo em formato de laudo

## Recursos de Voz

### Vocabulário Médico

O sistema reconhece termos técnicos de radiologia:
- Anatomia e estruturas
- Patologias e achados
- Medidas e dimensões
- Terminologia padrão

### Processamento Inteligente

Após o ditado, a IA:
- Estrutura o texto em seções apropriadas
- Corrige erros de transcrição
- Aplica formatação padrão
- Sugere conclusões baseadas nos achados

## Atalhos de Voz

| Atalho | Função |
|--------|--------|
| `Espaço` | Iniciar/parar ditado (editor vazio) |
| `Alt + M` | Iniciar/parar ditado (qualquer momento) |
| `Cmd + S` | Salvar laudo |
| `Cmd + K` | Buscar laudos |
| `Cmd + /` | Ver todos os atalhos |
