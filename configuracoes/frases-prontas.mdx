# Frases Prontas

As Frases Prontas sao textos padronizados que a IA utiliza literalmente ao gerar laudos, garantindo consistencia na redacao.

## Acessando

Navegue para **Configuracoes > Frases Prontas** ou `/dashboard/memory`

## Estrutura das Frases

### Colunas da Tabela

| Coluna | Descricao |
|--------|-----------|
| **Categoria** | Classificacao da frase |
| **Frase** | Texto completo da frase |
| **Palavras-chave** | Tags que ativam a frase |
| **Usos** | Quantidade de vezes utilizada |
| **Data** | Data de criacao |

## Criar Nova Frase

<Steps>
  <Step title="Clique em Nova Frase">
    Use o botao **+ Nova Frase** no canto superior direito
  </Step>

  <Step title="Defina a Categoria">
    Escolha ou crie uma categoria para organizar a frase
  </Step>

  <Step title="Digite a Frase">
    Escreva o texto exato que a IA deve usar
  </Step>

  <Step title="Adicione Palavras-chave">
    Defina tags que ativarao o uso desta frase
  </Step>

  <Step title="Salve">
    Confirme para adicionar a sua biblioteca
  </Step>
</Steps>

## Filtros

- **Todas categorias**: Visualizar todas as frases
- **Filtro por categoria**: Selecionar categoria especifica
- **Busca**: Pesquisar por texto ou palavra-chave
