# Preferências

Acesse **Configurações > Preferências** ou `/settings` para personalizar a plataforma.

## Aparência

Escolha o tema da interface:

| Tema | Descrição |
|------|-----------|
| **Claro** | Interface com fundo branco |
| **Escuro** | Interface com fundo escuro (padrão) |
| **Sistema** | Segue configuração do sistema operacional |

## Editor

<ParamField body="Salvamento automático" type="toggle">
  Quando ativado, salva laudos automaticamente durante a edição.
</ParamField>

<ParamField body="Copiar ao gerar" type="toggle">
  Quando ativado, copia automaticamente o laudo para a área de transferência após geração.
</ParamField>

## Geração de Laudos

<ParamField body="Frases Prontas" type="toggle">
  Quando ativado, a IA adapta suas frases personalizadas ao laudo.
</ParamField>

### Gerenciar Frases

Link rápido para adicionar e editar frases que a IA usa nos laudos.

## Integrações

### PACS e EHR

Configure conexões com seu sistema PACS e prontuário eletrônico.

<Card title="PACS e EHR" icon="hospital">
  Configure conexões com seu PACS e prontuário
</Card>

<Info>
A integração com PACS permite:
- Importar exames diretamente para criação de laudos
- Enviar laudos finalizados automaticamente
- Sincronizar dados do paciente
</Info>

<Note>
**Disponível para planos Enterprise.** [Fale com vendas](/configuracoes/planos) para configurar integrações personalizadas.
</Note>
