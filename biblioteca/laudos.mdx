# Biblioteca de Laudos

Acesse todos os seus laudos criados e gerencie-os de forma centralizada.

## Navegação

Acesse via menu lateral: **Biblioteca > Laudos** ou pela URL `/dashboard/reports`

## Interface da Biblioteca

### Filtros Disponíveis

#### Filtro por Período
- **Todos** - Todos os laudos (padrão)
- **Hoje** - Apenas laudos de hoje
- **Última semana** - Laudos dos últimos 7 dias
- **Último mês** - Laudos dos últimos 30 dias

#### Filtro por Organização
- **Todas as organizações** - Exibe laudos de todas as organizações vinculadas

### Pesquisa

Use o campo de busca **"Pesquisar laudos..."** para localizar laudos por título ou conteúdo.

<Tip>
A busca funciona em **tempo real** - os resultados são filtrados conforme você digita. Você pode buscar por:
- Nome do exame (ex: "Tomografia")
- Tipo de laudo (ex: "TC ABD")
- Data específica
</Tip>

### Tabela de Laudos

| Coluna | Descrição |
|--------|-----------|
| **Título** | Nome do laudo com data |
| **Tipo** | Categoria ou modelo usado |
| **Data** | Data de criação |
| **Versão** | Versão do laudo (v1, v2, etc.) |
| **Ações** | Editar, opções adicionais |

## Tipos de Laudos

<AccordionGroup>
  <Accordion title="Laudo por Voz">
    Laudos criados através de ditado por voz, marcados com tags "Voz" e "Auto-gerado"
  </Accordion>

  <Accordion title="Laudo por Modelo">
    Laudos baseados em templates, exibem o nome do modelo utilizado
  </Accordion>

  <Accordion title="Laudo Manual">
    Laudos digitados manualmente sem uso de templates
  </Accordion>
</AccordionGroup>

## Ações Disponíveis

- **Novo Laudo**: Abre o editor para criar novo laudo
- **Editar**: Abre o laudo no editor
- **Menu de opções**: Ações adicionais por laudo

## Paginação

A biblioteca exibe 50 laudos por página com navegação:
- Botões Anterior/Próxima
- Indicador de página atual (ex: Página 1 de 26)
- Total de laudos (ex: Mostrando 1 a 50 de 1300 laudos)
