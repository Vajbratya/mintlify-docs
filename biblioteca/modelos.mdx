# Modelos de Laudo

Os modelos (templates) permitem padronizar laudos e acelerar o processo de criacao.

## Acessando Modelos

Navegue para **Biblioteca > Modelos** ou acesse `/dashboard/templates`

## Tabela de Modelos

| Coluna | Descricao |
|--------|-----------|
| **Nome** | Nome identificador do modelo |
| **Usado** | Quantidade de vezes utilizado |
| **Categoria** | Classificacao (TC, RM, USG, etc.) |
| **Tipo** | Sistema ou Pessoal |
| **Curtidas** | Quantidade de curtidas da comunidade |

## Acoes

### Criar Novo Modelo

Clique em **+ Criar Modelo** para criar um template do zero.

### Importar de Imagem

Use **Importar de Imagem** para criar modelos a partir de laudos em imagem.

### Usar Modelo

Clique em **Usar** para abrir o editor com o modelo selecionado.

## Categorias Comuns

- **TC**: Tomografia Computadorizada (Cranio, Face, Abdome, Torax)
- **RM**: Ressonancia Magnetica
- **USG**: Ultrassonografia
- **TCPS**: TC de Corpo Inteiro
- **ANGIOTC**: Angiotomografia
