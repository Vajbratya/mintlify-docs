# Inicio Rapido

Siga estes passos para criar seu primeiro laudo com assistencia de IA.

## 1. Acesse a Plataforma

Navegue para [copilot.laudos.ai](https://copilot.laudos.ai) e faca login com sua conta.

## 2. Abra o Editor

Apos o login, voce sera direcionado ao **Editor** - a interface principal de criacao de laudos.

## 3. Escolha como Criar seu Laudo

<CardGroup cols={2}>
  <Card title="Ditar Laudo" icon="microphone">
    Pressione **Espaco** para iniciar o ditado por voz. Fale naturalmente em portugues.
  </Card>
  <Card title="Usar Modelo" icon="file-lines">
    Selecione um template da sua biblioteca de modelos frequentes.
  </Card>
  <Card title="Digitar" icon="keyboard">
    Comece a digitar diretamente no editor com assistencia de IA.
  </Card>
  <Card title="Em Branco" icon="file">
    Inicie um laudo completamente do zero.
  </Card>
</CardGroup>

## 4. Edite e Formate

Use a barra de ferramentas para formatar seu laudo:

| Atalho | Funcao |
|--------|--------|
| `Cmd + B` | Negrito |
| `Cmd + I` | Italico |
| `Cmd + U` | Sublinhado |
| `Cmd + S` | Salvar |

## 5. Salve e Publique

- O **salvamento automatico** esta ativado por padrao
- Clique em **Publicar** para finalizar o laudo
- Use **Copiar** para enviar ao seu sistema PACS/RIS

<Tip>
Ative "Copiar ao gerar" nas configuracoes para copiar automaticamente cada laudo apos a geracao.
</Tip>
