# Início Rápido

Siga estes passos para criar seu primeiro laudo com assistência de IA.

## 1. Acesse a Plataforma

Navegue para [copilot.laudos.ai](https://copilot.laudos.ai) e faça login com sua conta.

## 2. Abra o Editor

Após o login, você será direcionado ao **Editor** - a interface principal de criação de laudos.

## 3. Escolha como Criar seu Laudo

<CardGroup cols={2}>
  <Card title="Ditar Laudo" icon="microphone">
    Pressione **Espaço** para iniciar o ditado por voz. Fale naturalmente em português.
  </Card>
  <Card title="Usar Modelo" icon="file-lines">
    Selecione um template da sua biblioteca de modelos frequentes.
  </Card>
  <Card title="Digitar" icon="keyboard">
    Comece a digitar diretamente no editor com assistência de IA.
  </Card>
  <Card title="Em Branco" icon="file">
    Inicie um laudo completamente do zero.
  </Card>
</CardGroup>

## 4. Edite e Formate

Use a barra de ferramentas para formatar seu laudo:

| Atalho | Função |
|--------|--------|
| `Cmd + B` | Negrito |
| `Cmd + I` | Itálico |
| `Cmd + U` | Sublinhado |
| `Cmd + S` | Salvar |
| `Cmd + K` | Busca global |
| `Cmd + N` | Novo laudo |
| `Alt + M` | Ditado por voz |

## 5. Salve e Publique

- O **salvamento automático** está ativado por padrão
- Clique em **Publicar** para finalizar o laudo
- Use **Copiar** para enviar ao seu sistema PACS/RIS

<Tip>
Ative "Copiar ao gerar" nas configurações para copiar automaticamente cada laudo após a geração.
</Tip>
