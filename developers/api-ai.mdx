# AI Generation API

Endpoint para gerar laudos radiologicos usando IA.

## Base URL

```
https://copilot.laudos.ai/api/v1
```

---

## Gerar Laudo

<ParamField path="POST" method="/ai/generate">
  Gera um laudo radiologico a partir de achados ou historico clinico.
</ParamField>

### Body Parameters

| Campo | Tipo | Obrigatorio | Descricao |
|-------|------|-------------|-----------|
| `findings` | string | Sim* | Achados do exame |
| `clinical_history` | string | Sim* | Historico clinico |
| `exam_type` | string | Nao | Tipo de exame |
| `modality` | string | Nao | Modalidade (CT, MR, US, etc.) |
| `template_id` | string | Nao | ID do template a usar |
| `patient_age` | string | Nao | Idade do paciente |
| `patient_gender` | string | Nao | Genero (M/F) |
| `indication` | string | Nao | Indicacao clinica |
| `format` | string | Nao | Formato de saida: `html` (padrao) ou `text` |

<Note>
  Pelo menos um dos campos `findings` ou `clinical_history` e obrigatorio.
</Note>

### Exemplo

```bash
curl -X POST "https://copilot.laudos.ai/api/v1/ai/generate" \
  -H "Authorization: Bearer sk_live_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "findings": "Nodulo pulmonar de 8mm no lobo superior direito",
    "exam_type": "CT",
    "modality": "CT",
    "patient_age": "65 anos",
    "patient_gender": "M",
    "indication": "Rastreamento de cancer de pulmao"
  }'
```

### Resposta

```json
{
  "data": {
    "content": "<h1>LAUDO RADIOLOGICO</h1><h2>TC DE TORAX</h2><p><strong>Tecnica:</strong> Tomografia computadorizada...</p><p><strong>Achados:</strong> Nodulo pulmonar solido...</p><p><strong>Impressao:</strong> Nodulo pulmonar...</p>",
    "format": "html",
    "model": "gpt-5",
    "usage": {
      "prompt_tokens": 150,
      "completion_tokens": 500,
      "total_tokens": 650
    }
  }
}
```

---

## Timeout

A geracao de laudos com IA pode levar ate 2 minutos. O endpoint possui timeout de 120 segundos.

---

## Rate Limits

| Plano | Requests/min |
|-------|--------------|
| Free | 5 |
| Pro | 30 |
| Enterprise | 150 |
| Unlimited | Sem limite |

---

## Codigos de Erro

| Codigo | Descricao |
|--------|-----------|
| 400 | `findings` ou `clinical_history` obrigatorio |
| 401 | Nao autenticado |
| 403 | Sem permissao para AI |
| 429 | Rate limit excedido |
| 500 | Erro na geracao |
| 504 | Timeout (> 120s) |
