# SDKs Oficiais

Oferecemos SDKs para as principais linguagens, simplificando a integração.

## Instalação

<CodeGroup>

```bash Node.js
npm install @laudosai/sdk
# ou
yarn add @laudosai/sdk
```

```bash Python
pip install laudosai
```

</CodeGroup>

## Configuração

<CodeGroup>

```javascript Node.js
import { LaudosAI } from '@laudosai/sdk';

const laudos = new LaudosAI({
  apiKey: process.env.LAUDOSAI_API_KEY,
  environment: 'production' // ou 'sandbox'
});
```

```python Python
from laudosai import LaudosAI

laudos = LaudosAI(
    api_key=os.environ['LAUDOSAI_API_KEY'],
    environment='production'  # ou 'sandbox'
)
```

</CodeGroup>

## Exemplos

### Enviar Laudo ao PACS

<CodeGroup>

```javascript Node.js
// Enviar laudo para conexão PACS configurada
const result = await laudos.pacs.sendReport({
  connectionId: 'conn_xxxxx',
  studyInstanceUID: '1.2.840.113619.xxx',
  reportText: `
    TÉCNICA: RM de crânio sem contraste.

    ANÁLISE: Parênquima cerebral com morfologia e sinal preservados...

    CONCLUSÃO: Exame dentro dos limites da normalidade.
  `,
  accessionNumber: 'ACC12345',
  modality: 'MR'
});

if (result.success) {
  console.log('Laudo enviado:', result.sopInstanceUID);
} else {
  console.error('Erro:', result.error);
}
```

```python Python
# Enviar laudo para conexão PACS configurada
result = laudos.pacs.send_report(
    connection_id='conn_xxxxx',
    study_instance_uid='1.2.840.113619.xxx',
    report_text="""
    TÉCNICA: RM de crânio sem contraste.

    ANÁLISE: Parênquima cerebral com morfologia e sinal preservados...

    CONCLUSÃO: Exame dentro dos limites da normalidade.
    """,
    accession_number='ACC12345',
    modality='MR'
)

if result.success:
    print(f'Laudo enviado: {result.sop_instance_uid}')
else:
    print(f'Erro: {result.error}')
```

</CodeGroup>

### Testar Conexão PACS

<CodeGroup>

```javascript Node.js
const test = await laudos.pacs.testConnection('conn_xxxxx');

console.log({
  connected: test.success,
  latency: `${test.latencyMs}ms`,
  server: test.serverInfo?.server
});
```

```python Python
test = laudos.pacs.test_connection('conn_xxxxx')

print({
    'connected': test.success,
    'latency': f'{test.latency_ms}ms',
    'server': test.server_info.get('server')
})
```

</CodeGroup>

### Listar Conexões

<CodeGroup>

```javascript Node.js
const connections = await laudos.pacs.list();

for (const conn of connections) {
  console.log(`${conn.name}: ${conn.status}`);
}
```

```python Python
connections = laudos.pacs.list()

for conn in connections:
    print(f'{conn.name}: {conn.status}')
```

</CodeGroup>

### Webhooks

<CodeGroup>

```javascript Node.js
// Criar webhook
const webhook = await laudos.webhooks.create({
  url: 'https://your-app.com/webhooks',
  events: ['report.finalized', 'critical_finding.detected'],
  secret: 'whsec_xxxxx'
});

// Helper para verificar assinatura
import { verifyWebhookSignature } from '@laudosai/sdk';

app.post('/webhooks', (req, res) => {
  const isValid = verifyWebhookSignature(
    req.body,
    req.headers['x-laudos-signature'],
    process.env.WEBHOOK_SECRET
  );

  if (!isValid) return res.status(401).send('Invalid');

  // Processar...
  res.send('OK');
});
```

```python Python
# Criar webhook
webhook = laudos.webhooks.create(
    url='https://your-app.com/webhooks',
    events=['report.finalized', 'critical_finding.detected'],
    secret='whsec_xxxxx'
)

# Helper para verificar assinatura
from laudosai import verify_webhook_signature

@app.route('/webhooks', methods=['POST'])
def handle_webhook():
    is_valid = verify_webhook_signature(
        request.data,
        request.headers['X-Laudos-Signature'],
        os.environ['WEBHOOK_SECRET']
    )

    if not is_valid:
        return 'Invalid', 401

    # Processar...
    return 'OK'
```

</CodeGroup>
