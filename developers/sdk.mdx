# @laudosai/sdk

SDK oficial TypeScript para integracao com a API do Laudos.AI. Type-safe, compativel com Node.js 18+, Bun e Deno.

<Note>
  O SDK esta em desenvolvimento. Os exemplos abaixo mostram a API planejada. Enquanto o SDK nao estiver publicado, use `fetch` diretamente conforme mostrado na documentacao de cada endpoint.
</Note>

---

## Instalacao

<CodeGroup>

```bash npm
npm install @laudosai/sdk
```

```bash pnpm
pnpm add @laudosai/sdk
```

```bash bun
bun add @laudosai/sdk
```

</CodeGroup>

## Quick Start

```typescript
import { LaudosAI } from '@laudosai/sdk';

const client = new LaudosAI({
  apiKey: process.env.LAUDOSAI_API_KEY,
});

// Listar laudos
const { data: reports } = await client.reports.list({ limit: 10 });
console.log(`${reports.length} laudos encontrados`);

// Criar laudo
const { data: report } = await client.reports.create({
  title: 'TC de Torax',
  content: '<h1>LAUDO RADIOLOGICO</h1><p>Exame normal.</p>',
  exam_type: 'CT',
});
console.log(`Laudo criado: ${report.id}`);
```

## Configuracao

```typescript
const client = new LaudosAI({
  apiKey: process.env.LAUDOSAI_API_KEY,

  // Timeout em milissegundos (default: 30000)
  timeout: 60000,

  // Numero maximo de retries (default: 2)
  maxRetries: 3,
});
```

## Reports

```typescript
// Listar
const { data, meta } = await client.reports.list({
  limit: 20,
  exam_type: 'CT',
  sort: 'desc',
});

// Obter por ID
const { data: report } = await client.reports.get('rpt_abc123');

// Criar
const { data: newReport } = await client.reports.create({
  title: 'TC de Torax',
  content: '<h1>LAUDO RADIOLOGICO</h1><p>Exame normal.</p>',
  exam_type: 'CT',
  accession_number: 'ACC2026012600001',
  patient_id: 'PAT001',
  patient_name: 'Silva, Joao',
  status: 'final',
});

// Atualizar
const { data: updated } = await client.reports.update('rpt_abc123', {
  title: 'TC de Torax - Corrigido',
  status: 'corrected',
});

// Excluir
await client.reports.delete('rpt_abc123');
```

## Workflow Completo

Exemplo: Criar laudo e enviar ao PACS.

```typescript
import { LaudosAI } from '@laudosai/sdk';

const client = new LaudosAI({
  apiKey: process.env.LAUDOSAI_API_KEY,
  timeout: 120000,
});

// 1. Criar o laudo
const { data: report } = await client.reports.create({
  title: 'TC de Torax',
  content: laudoHTML,
  exam_type: 'CT',
  accession_number: 'ACC2026012600001',
  patient_id: 'PAT001',
  patient_name: 'Silva, Joao Carlos',
  status: 'final',
});

console.log(`Laudo criado: ${report.id}`);

// 2. Enviar ao PACS
const result = await client.pacs.send({
  connection_id: 'conn_abc123',
  report_text: laudoHTML,
  report_status: 'final',
  accession_number: 'ACC2026012600001',
  patient_id: 'PAT001',
  patient_name: 'Silva, Joao Carlos',
  modality: 'CT',
});

if (result.success) {
  console.log(`Enviado ao PACS via ${result.protocol}`);
}
```

## Tratamento de Erros

```typescript
import { LaudosAI, LaudosAIError } from '@laudosai/sdk';

try {
  await client.reports.get('invalid-id');
} catch (error) {
  if (error instanceof LaudosAIError) {
    console.error(`Status: ${error.status}`);
    console.error(`Mensagem: ${error.message}`);

    if (error.status === 401) {
      console.error('Verifique sua API key');
    }
    if (error.status === 429) {
      console.error(`Rate limit. Retry apos ${error.retryAfter}s`);
    }
  }
}
```
